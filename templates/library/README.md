# My Knowledge Library

A personal knowledge library powered by [stacks](https://github.com/chuggies510/stacks).

## Stacks

See [catalog.md](catalog.md) for the full index.

## Usage

- Drop sources in `{stack}/sources/incoming/`
- Run `/stacks:ingest {stack}` to process them
- Run `/stacks:refine {stack}` to check coverage and find gaps
- Run `/stacks:lookup "your question"` from any repo to query
